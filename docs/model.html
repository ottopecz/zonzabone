Model

General

A model is an enhanced javascript object. The added methods operate on the core object of the model.

get (method)

model.get(key)

Returns the value of the property with the given key.

params

  • key: The key of the property.

returns

The value of the property with the given key.

example usage

var value = model.get("someKey")

set (method)

model.set(key, value)

Sets the property with the given key to the given value

params

  • key: The key of the property we want to set.
  • value: The value of the property we want to set.

returns

The context model.

example usage

model.set("someKey": "someValue")

core (method)

model.core()

Returns the core object of the model.

params

None

returns

The core object of the model.

example usage

var obj = model.core();

init (method)

model.init(options)

Sets up the model. Any passed arguments will enter the core object of the model. If the argument is an array of valid objects it will be automatically wrapped up into a collection

params

  • options: The object of the initial arguments

returns

The context model

example usage

model.init({ "arrOfObj": [{ "k1": "v1" }, { "k2": "v2" }], "someOtherArg": true })

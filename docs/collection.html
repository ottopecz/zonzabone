Collection

A collection is a list of native javascript objects

General

A collection is a list of native javascript objects. The factory function of the collection doesn't check on the type of the initial arguments so you have to make sure that it's really an array of objects.

reset (method)

collection.reset(arr)

Resets the core of the collection either to an empty array or the passed array

params

  • arr: !!!optional!!! The new core array of the collection. If it's left undefined the new core will be an empty array

returns

The context collection

example usage

coll.reset([{"k1": "v1"}, {"k2": "v2"}])

length (property)

collection.length

The length (number of elements) of the collection. This is a proxy to the native length property of the core array.

type

Number

example usage

var l = coll.length;

core (method)

collection.core()

Returns the core array of the collection

params

  • No argument needed

returns

An core array of the collection

example usage

var arr = coll.core()

get (method)

collection.get(key)

Returns all the element with the given key

params

  • key: The common key of the elements we want to retrieve.

returns

Either an array of the hits or - if there is only one result - an object

example usage

var ret = coll.get("robinTheBirdy")

set (method)

collection.set(key, value)

Sets the given key, value pair on every element in the collection. (Doesn't check on the key beforehand. So if the key is already set it's gonna be overwritten)

params

  • key: the key property to set on every element
  • value: the value of the key property to set on every element

returns

The context collection

example usage

collection.get("robinTheBirdy", "singsNicely")

add (method)

collection.add(arr||obj)

Adds a new array or a single object to the collection.

params

  • arr||obj: The array or obj to add to the collection

returns

Nothing at this moment but probably that's a mistake.

example usage

coll.add([{"k1": "v1"}, {"k2": "v2"}])

removeWhere (method)

collection.removeWhere(filters||key, value)

Removes those elements from the collection which have all the key value pair given as arguments

params

  • filters: An object the key, value pairs we want to filter with

or

  • key: The key we want to filter with
  • value: The value of the key we want to filter with

returns

The array of the removed elements.

example usage

coll.removeWhere({"k1": "v1", "k2": "v2"})

or

coll.removeWhere("k1", "v1")

getWhere (method)

collection.getWhere(key, value)

Returns all the elements which have the given key value pair.

params

  • key: The key to filter with.
  • value: The value of the key to filter with.

returns

The array of result elements

example usage

var arr = coll.getWhere("someKey", "someValue")

without (method)

collection.getWhere(b, func)

Gets new array of all items in this collection and not in b. Relative complement of B in A

params

  • b: Collection of containing objects to filter from collection.
  • func: Comparator function to determine object match.

example usage

coll1.without(coll2, function (coll1, coll2) { return coll1.prop === coll2.prop; })

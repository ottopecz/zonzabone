View Collection

General

A collection of native javascript objects. This type of collection extends the plain collection see collection The addition is the capability of fetching data from the server. You can specify custom ajax parameters either upon construction like var viewCollection = collection({"url": "/foo/bar/", "ajaxSuccess": function (coll) {}}) or when you call fetch see fetch. If custom ajax options are passed for both the constructor and the fetch method the fetch ones override the constructor ones. This type is intended to be the data structure of list views.
